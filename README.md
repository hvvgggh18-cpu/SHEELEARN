# SHEELEARN

<p align="center">
  <img src="public/favicon.ico" alt="SHEELEARN Logo" width="120">

<p align="center">
An AI-powered learning platform that transforms notes, PDFs, and study materials into summaries, quizzes, flashcards, essays, study plans, and intelligent conversations.
</p>

---

## 🚀 Features

- 🤖 AI Chat Tutor
- 📄 AI Summarizer
- 🧠 Flashcard Generator
- 📝 Quiz Generator
- 📚 Study Planner
- 📂 Document Management
- 📊 Learning Analytics
- 🗒️ Smart Notes
- 🔐 Secure Authentication
- 🌙 Modern Dark UI
- 📱 Progressive Web App (PWA)

---

## 🛠️ Tech Stack

- Laravel 12
- PHP 8.3+
- MySQL
- Blade
- Tailwind CSS
- Alpine.js
- JavaScript
- OpenAI / LLM API
- Laravel Queues

---

## 📦 Installation

```bash
git clone https://github.com/hvvgggh18-cpu/SHEELEARN.git

cd SHEELEARN

composer install

npm install

cp .env.example .env

php artisan key:generate

php artisan migrate

npm run dev

php artisan serve
```
